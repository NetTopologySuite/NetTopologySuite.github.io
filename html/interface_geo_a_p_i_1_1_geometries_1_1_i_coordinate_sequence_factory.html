ICoordinateSequence GeoAPI.Geometries.ICoordinateSequenceFactory.Create ( int  size,
int  dimension 
)

Creates a ICoordinateSequence of the specified size and dimension. For this to be useful, the ICoordinateSequence implementation must be mutable.

If the requested dimension is larger than the CoordinateSequence implementation can provide, then a sequence of maximum possible dimension should be created. An error should not be thrown.

Parameters
size
dimensionthe dimension of the coordinates in the sequence (if user-specifiable, otherwise ignored)
Returns
A coordinate sequence
