Detailed Description

Creates spatial reference objects using codes.

The codes are maintained by an external authority. A commonly used authority is EPSG, which is also used in the GeoTIFF standard and in SharpMap.

string GeoAPI.CoordinateSystems.ICoordinateSystemAuthorityFactory.GeoidFromWktName ( string  wkt)

Gets the Geoid code from a WKT name.

In the OGC definition of WKT horizontal datums, the geoid is referenced by a quoted string, which is used as a key value. This method converts the key value string into a code recognized by this authority.
