Detailed Description

A lightweight class used to store coordinates on the 2-dimensional Cartesian plane.

It is distinct from IPoint, which is a subclass of IGeometry. Unlike objects of type IPoint (which contain additional information such as an envelope, a precision model, and spatial reference system information), a <other>Coordinate</other> only contains ordinate values and propertied.

<other>Coordinate</other>s are two-dimensional points, with an additional Z-ordinate. If an Z-ordinate value is not specified or not defined, constructed coordinates have a Z-ordinate of NaN (which is also the value of NullOrdinate).

Apart from the basic accessor functions, NTS supports only specific operations involving the Z-ordinate.
