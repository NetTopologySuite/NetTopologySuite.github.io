Detailed Description

The internal representation of a list of coordinates inside a Geometry.

This allows Geometries to store their points using something other than the NTS Coordinate class. For example, a storage-efficient implementation might store coordinate sequences as an array of x's and an array of y's. Or a custom coordinate class might support extra attributes like M-values.

Implementing a custom coordinate storage structure requires implementing the ICoordinateSequence and ICoordinateSequenceFactory interfaces. To use the custom CoordinateSequence, create a new IGeometryFactory parameterized by the CoordinateSequenceFactory The IGeometryFactory can then be used to create new IGeometrys. The new Geometries will use the custom CoordinateSequence implementation.

Returns the ordinate of a coordinate in this sequence. Ordinate indices 0 and 1 are assumed to be X and Y. Ordinate indices greater than 1 have user-defined semantics (for instance, they may contain other dimensions or measure values).

If the sequence does not provide value for the required ordinate, the implementation must not throw an exception, it should return Coordinate.NullOrdinate.
